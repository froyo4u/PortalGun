# Portal Gun mod

A Minecraft mod that provides portal gun using [Immersive Portals](https://github.com/iPortalTeam/ImmersivePortalsMod) functionality.

It's a fork of [MeowMC's Portal Gun mod](https://github.com/MeowMC/PortalGun).
